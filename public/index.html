Generate, populate, and validate forms via JSON and JSON Schema

Last updated: 12.02.2025, uses handlebars.js and ajv.js

The form posts data to a serverless function which then writes to a MongoDB database. Form data is validated both in browser and on the server using the same JSON Schema. For sake of this test browser's default validation has been disabled using 'novalidate' attribute on form. Code is available on GitHub
