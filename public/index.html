Circle of Data

This is an example of transfering data between browser and server using JavaScript without any frameworks. When this page loads, JS fetches this json which contains information to render the form. JS then feeds json data into this handlebars template to generate HTML for the form which is then added to the page. When form is submitted, JS posts form data to a Node.js endpoint which writes data to MongoDB database. Data in database is fetched by JS in browser calling this endpoint and displaying results on the page.
