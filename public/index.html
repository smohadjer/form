Circle of Data

This is an example of transferring data between browser and server using JavaScript without any frameworks. When the page loads in browser, JavaScript fetches this json which holds form fields. JavaScript then feeds the json data into this Handlebars template to generate HTML for the form which is then added to the page. When user submits the form, JavaScript sends a post request to a Node.js function which writes the data to a MongoDB database. Data in database is fetched and displayed in browser via JavaScript calling this API endpoint.
