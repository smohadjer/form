Circle of Data

This demo showcases the following features:

  • Generating HTML Forms dynamically from static JSON using JavaScript and Handlebars library
  • Client-side and Server-side form validation
  • Read and Write data from/to Database via serverless function

Description:

The form on this page is generated by JavaScript from a static json and a Handlebars template which gets populated by data from database on page load. When form is submitted, JavaScript validates the form and sends data to api endpoint which then validates data again and upon validation success writes it to database. The data in database is fetched via this API endpoint and displayed on the page. Code available on GitHub
